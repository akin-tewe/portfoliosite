### Liquid Glass System (Research card only)

Constants:
  GLASS_MAP_SIZE = 256        — canvas resolution for displacement/specular maps
  GLASS_DOT_SIZE = 150        — rendered dot diameter in px
  GLASS_BEZEL_RATIO = 0.4     — proportion of radius that forms the refractive edge
  GLASS_N1 = 1.0, GLASS_N2 = 1.5  — refractive indices (air → glass)
  GLASS_SCALE = 35            — feDisplacementMap scale value
  GLASS_SURFACE              — quartic surface profile function

Functions:
  generateDisplacementMap()   — Canvas → data URL. Snell's law refraction table (127 samples),
                                radial SDF, encodes normal vectors as R/G channels (128 = neutral).
  generateSpecularFillMap()   — Canvas → data URL. Dual light sources (upper-left + lower-right),
                                cubic edge falloff, 18% milky fill + rim specular. Purple-free
                                (neutral white tones only).

Components:
  GlassDot                    — 150×150 circle with backdrop-filter: url(#glass-circle)
  DotField                    — 6 fixed-slot layout with motion.div position/scale animation.
                                Layouts keyed by dot count (1-5), nulls scale to 0.
  ResearchCardOverlay (memo)  — Combines ShaderGradient background + cycling Maslow level text
                                (instrumentSerif, 3.3s interval) + SVG filter + DotField.

SVG filter pipeline (id="glass-circle"):
  feImage(disp) → feDisplacementMap(scale 35) → feColorMatrix(saturate 1.3)
  → feImage(spec) → feBlend(screen)

### Maslow Levels

  5 dots → "physiological"
  4 dots → "safety"
  3 dots → "belonging"
  2 dots → "esteem"
  1 dot  → "self-actualization"

Cycles every 3.3 seconds. Text rendered in Instrument Serif (instrumentSerif),
text-8xl md:text-9xl, text-gray-900/15, lowercase.

### Card Rendering Logic

Each card in ProjectsGrid:
  - locked? → non-clickable div with ClockAlert icon + dimmed pill
  - else → HoverCard (Link + cursor-following modal)

Inside card:
  - gradient? → ShaderGradientCanvas (pixelated mode optional)
  - id===2? → ResearchCardOverlay
  - id===6? → ChatPillsOverlay
  - image? → centered img (max-w-[60%] max-h-[60%])
  - video? → LazyVideo
  - slideshow? → id===3 uses crossfade + film grain; others use vertical slide + vignette

### Hover Modal System

HoverCard wraps each project Link. On hover, a fixed-position modal follows
the cursor using requestAnimationFrame + calcModalPosition (viewport-aware
edge clamping). Modal contains: title (Pixelify), body (Roboto), tag pills
(color-coded from TAG_COLORS map).

3. VIDEO COMPONENTS: components/SplashVideo.tsx
================================================================================

Two video components:

TransparentVideo (default export)
  Props: { mp4Src?: string, webmSrc?: string }
  - Absolute positioned, object-contain
  - mp4 source uses type="video/mp4;codecs=hvc1" (Safari HEVC alpha)
  - webm source uses type="video/webm" (Chrome/Firefox VP9 alpha)
  - autoPlay, loop, muted, playsInline

MainVideo (named export)
  Props: { webmSrc: string, mp4Src?: string }
  - Desktop only (useMinWidth(1024) gate)
  - Same dual-source pattern (HVC1 + webm)
  - Used for hero splash video

NOTE: TransparentVideo was modified to accept webmSrc (task 111). Previously
it only took mp4Src with type="video/mp4". Both components now support the
dual-source transparent video pattern.
