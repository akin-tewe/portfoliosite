ANIMATION PATTERN:
------------------
All scroll-triggered animations follow this pattern:

1. Import hooks:
   import { motion, useInView } from "framer-motion"
   import { useRef } from "react"

2. Create ref and detect visibility:
   const ref = useRef(null)
   const isInView = useInView(ref, { once: true, margin: "-100px" })

3. Apply animation:
   <motion.div
     ref={ref}
     initial={{ opacity: 0, y: 30 }}
     animate={isInView ? { opacity: 1, y: 0 } : {}}
     transition={{ duration: 0.6 }}
   >

STAGGER PATTERN:
----------------
For staggered children animations:

const containerVariants = {
  hidden: { opacity: 0 },
  visible: {
    opacity: 1,
    transition: {
      staggerChildren: 0.1,
      delayChildren: 0.2
    }
  }
}

const itemVariants = {
  hidden: { opacity: 0, y: 20 },
  visible: { opacity: 1, y: 0 }
}

CHART COMPONENT PATTERN:
------------------------
All chart components share:
- useInView for scroll triggering
- Responsive sizing with Tailwind breakpoints
- Consistent color palette from CHART_COLORS array
- maxValue prop for consistent scale across comparisons

MOBILE-FIRST BREAKPOINTS:
-------------------------
Base styles: mobile (< 768px)
md: prefix: tablet/desktop (>= 768px)
lg: prefix: large desktop (>= 1024px)
